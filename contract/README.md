# LET'S BUILD A CONTRACT

# SETUP
* ```nvm use```
* ```npm install```

# REMIX IDE

* ```npm run dev```
* ```navigate to http://localhost:8080```
* Select Environments -> Solidity
* Select File -> Connect to Localhost
* File explorers tab -> Open up a Contract
